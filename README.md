Unemployment Analysis with Python

Introduction

This project analyzes unemployment data in India using Python. The aim is to understand unemployment trends and visualize the data using graphs.

Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn

Dataset

The dataset contains information about:

- Region
- Date
- Unemployment Rate (%)
- Number of Employed People
- Labour Participation Rate (%)

Steps Performed

1. Loaded the dataset.
2. Cleaned the data.
3. Analyzed unemployment rates.
4. Created graphs and charts.
5. Studied unemployment trends during COVID-19.

Results

- Unemployment rates differ across states.
- COVID-19 increased unemployment in many regions.
- Graphs helped identify important trends and patterns.

Conclusion

Python makes it easy to analyze and visualize unemployment data. This project helps understand employment conditions in India and provides useful insights.

How to Run

1. Install required libraries:
   pip install pandas matplotlib seaborn

2. Run the program:
   python unemployment_analysis.py
